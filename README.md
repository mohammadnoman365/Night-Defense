# Night Defense

A 2D survival shooter game built in Unity where a dinosaur must survive 3 nights in a zombie apocalypse.

## About the Game
Night Defense is a 2D survival mobile game. The player controls a dinosaur and must survive 3 nights against waves of zombies. Each night has a time limit and limited bullets, so the player must shoot carefully to stay alive until morning.

## Gameplay
- Survive **3 nights** against zombie waves
- Each night has a **time limit**
- You have **limited bullets** so shoot carefully
- Kill zombies before they reach you

## Features
- 3 night survival levels
- Limited bullet 
- Night timer system
- Shooting mechanics
- Mobile-ready controls

## Built With
- **Engine:** Unity
- **Language:** C#
- **Platform:** Mobile (Android)
